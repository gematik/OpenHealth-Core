ParserScope

inner class ParserScope(startOffset: Int, endOffset: Int)

ParserScope implements the basic parsing functionality.

Important: ParserScope is treated as a mutable object throughout the entire time of calling read and therefore unsafe to keep references to it or use it in the context of coroutines or threads.
